# RoleRadar

Track job market trends. Discover in-demand skills. Make smarter career moves.

RoleRadar is an end-to-end data platform that collects job listings, processes and analyzes them using data engineering pipelines and natural language processing (NLP), and presents actionable insights through a modern web dashboard.

This project demonstrates full-stack development, data engineering, and data science skills by simulating a real-world data product.

---

## Problem

Job seekers often struggle to understand:
- Which skills are actually in demand
- How job trends are changing over time
- What they need to learn to stay competitive

---

## Solution

RoleRadar addresses this by:
- Collecting job market data
- Extracting key skills from job descriptions
- Analyzing trends and demand patterns
- Presenting insights in a clean, interactive dashboard

---

## Features

- Job data ingestion (web scraping or APIs)
- Data cleaning and transformation (ETL pipeline)
- Skill extraction using NLP
- Trend analysis (top skills, demand over time)
- Filtering by role, location, and experience
- Modern dashboard interface built with Next.js and Tailwind CSS

---

## Architecture

Scraper → Raw Data → ETL Pipeline → Database → API → Frontend Dashboard  
                                 ↓  
                          NLP / Analysis
